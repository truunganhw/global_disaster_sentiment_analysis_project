# global_disaster_sentiment_analysis_project

This project on sentiment analysis of disaster-related tweets is divided into three tasks. These tasks involve training, testing, and evaluating models using Deep Learning architectures such as BERT-base and BERTweet, as well as applying Large Language Models (LLMs) like Gemini.

Task 1: Classifying tweets as either "Disaster" or "Non-disaster."
Task 2: Identifying the specific type of disaster (if applicable).
Task 3: Determining the semantic intent of the tweet—specifically, whether it conveys "information provision," "emotional expression," or an "urgent plea for help."

This classification system enables researchers and disaster relief teams to utilize an automated tool that interprets social media text, thereby facilitating the detection of emergencies and ensuring timely relief efforts.

The project was inspired by Typhoon Yagi (which has since been renamed), a storm that claimed thousands of lives in Vietnam in 2024; this event highlighted the need for a data science initiative to help address the challenges posed by similar disasters.
